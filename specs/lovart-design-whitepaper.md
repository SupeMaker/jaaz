## 一、产品定位与概述

**Lovart.ai** 是由 LiblibAI 团队推出的 **AI 设计 Agent（智能体）**，定位为"用对话完成专业设计"。它不是单纯的图片生成器，也不是模板编辑器，而是一个**能理解设计意图、自动编排多模型、交付完整设计成果的智能协作者**。

- 官网：https://www.lovart.ai/
- 上线时间：2025 年正式上线
- 底层模型：GPT-4o Vision、Flux、Kling、Veo 3、Seedream、Gemini Imagen 3、Luma uni-1 等
- 支持语言：中文、英文、日文、韩文、法文、德文、西班牙文、葡萄牙文、俄文等

## 二、完整功能列表

### 2.1 核心功能

| 功能模块 | 说明 |
|---|---|
| **自然语言设计界面** | 用户用日常语言描述设计需求，AI 自动分析、拆解、调用模型执行 |
| **ChatCanvas 无限画布** | 聊天、参考图、生成结果、编辑动作沉淀在同一画布，保持上下文 |
| **多模态生成** | 图像、视频、音频、脚本、3D、品牌资产一站式生成 |
| **MCoT（思维链推理）** | AI 像创意总监一样分析业务上下文、目标受众、品牌需求，再拆分任务 |
| **Talk.Tab.Tune 工作流** | Talk（描述需求）→ Tab（点击选中元素）→ Tune（迭代精修） |
| **多 Agent 编排** | 接到任务后自动判断调用哪些子模型，并行执行，一致性校验 |

### 2.2 图像相关功能

| 功能 | 说明 |
|---|---|
| 文生图 | 文字描述生成图像（海报、包装、角色、产品图等） |

| SVG 矢量化 | 将位图转为 SVG 矢量图，无限缩放不失真 |

## 三、使用方式详解

### 3.1 注册流程

1. 访问 lovart.ai 官网
2. Google 账号或邮箱注册
3. 免费获得每日刷新积分试玩
4. 进入主界面

### 3.2 权限层级（订阅体系）

| 套餐 | 月费 | 积分/月 | 分辨率上限 | 商用授权 | 团队协作 |
|---|---|---|---|---|---|
| Free | $0 | 每日少量 | 1024×1024 | 无 | 无 |
| Starter | $19.93 | 1,500 | 1024×1024 | 无 | 无 |
| Basic | $39.95 | 4,600 | 2048×2048 | 有 | 无 |
| Pro | $89.97 | 10,990 | 2048×2048 | 有 | 3 席位 |
| Ultimate | $159.99 | 21,990 | 4096×4096 | 有 | 10+ 席位+API |

- 年付享 8 折
- 积分不跨月累积，充值积分永不过期
- 每日额外 100 刷新积分（付费用户）

### 3.3 关键操作路径

**路径 1：从首页创建项目**
1. 首页输入框 → 选择任务类型（图片生成/设计/品牌/插画/电商/视频）
2. 输入自然语言描述 → AI 分析并生成
3. 生成结果进入画布继续编辑

**路径 2：画布内迭代**
1. 画布中选中对象 → 弹出编辑工具条
2. 使用高清放大/去背景/橡皮/编辑元素/编辑文字/下载
3. 通过对话面板继续精修

**路径 3：品牌一致性工作流**
1. Settings → Brand → 设置品牌色/Logo/字体
2. 后续生成自动遵循品牌规范

### 3.4 快捷键

| 快捷键 | 功能 |

## 四、网页设计系统解析

### 4.1 色彩方案

| 用途 | 色值（推断） | 说明 |
|---|---|---|
| 主背景 | `#0A0A0A` ~ `#111111` | 深色系，科技感 |
| 卡片/面板 | `#1A1A2E` ~ `#16213E` | 深蓝灰，层次分明 |
| 主强调色 | `#6C63FF` / `#7C3AED` | 紫色系，品牌识别 |
| 次强调色 | `#00D4AA` / `#10B981` | 绿色/青色，CTA 按钮 |
| 文字主色 | `#FFFFFF` / `#F5F5F5` | 白色/浅灰 |
| 文字次色 | `#9CA3AF` / `#6B7280` | 中灰 |
| 渐变 | 紫蓝渐变 `#6C63FF→#3B82F6` | Hero 区域、按钮 |
| 边框 | `rgba(255,255,255,0.1)` | 微透明分隔 |

### 4.2 字体排版

| 用途 | 字体 | 字号/字重 |
|---|---|---|
| 大标题（Hero） | Inter / SF Pro Display | 48-72px / Bold (700) |
| 章节标题 | Inter | 24-32px / Semibold (600) |
| 小标题 | Inter | 18-20px / Medium (500) |
| 正文 | Inter | 14-16px / Regular (400) |
| 辅助文字 | Inter | 12-13px / Light (300) |
| 代码/Prompt | JetBrains Mono / Fira Code | 14px / Regular |

### 4.3 栅格布局逻辑

- **首页**：全宽 Hero → 居中内容区（max-width: 1200px）→ 特性卡片网格（2-3 列）
- **画布页**：三栏布局 — 左侧 Chat Panel（280-360px）+ 中间 Canvas（flex-1）+ 右侧工具面板（按需）
- **设置页**：左侧导航 + 右侧内容区
- **卡片网格**：4 列响应式（大屏）→ 2 列（中屏）→ 1 列（小屏）

### 4.4 交互动效反馈

| 场景 | 动效 |
|---|---|
| 页面进入 | Fade-in + Slide-up（0.3-0.5s ease） |
| 卡片 Hover | Scale(1.02) + Shadow 加深 + 边框发光 |
| 按钮 Hover | 背景色渐变过渡 + 微缩放 |
| 生成等待 | 骨架屏 / 脉冲动画 / 进度条 |
| 画布操作 | 平滑缩放/平移（CSS transform） |
| 对话消息 | 逐字打字机效果 |
| 模型切换 | 下拉面板滑入/滑出 |

### 4.5 响应式断点

| 断点 | 宽度 | 布局变化 |
|---|---|---|
| Desktop | ≥1280px | 完整三栏布局 |
| Laptop | 1024-1279px | Chat Panel 可折叠 |
| Tablet | 768-1023px | 双栏，画布全屏 |
| Mobile | <768px | 单栏，底部输入框 |

## 六、与 Jaaz 项目的映射关系

| Lovart 功能 | Jaaz 现有对应 | 差距/需补充 |
|---|---|---|
| 首页 Hero + 任务入口 | `routes/index.tsx` + `CanvasList` | 需增加任务类型选择器（图片/设计/品牌/插画/电商/视频） |
| ChatCanvas 画布 | `CanvasExcali` (Excalidraw) + `Chat.tsx` | 已有基础，需增强 PopBar 工具条 |
| 对话式设计 | `ChatTextarea` + `sendMessages` | 已有，需增加 Talk.Tab.Tune 交互模式 |
| 模型选择器 | `ModelSelectorV2/V3` | 已有，需适配 Lovart 风格 UI |
| 品牌设置 | `settings/dialog` | 需增加 Brand 配置面板 |
| 多模态输出 | 部分视频支持 (`VideoElement`) | 需增加音频/3D/脚本生成入口 |
| 积分/订阅 | `PointsDisplay` + `billing.ts` | 已有基础，需对齐 Lovart 定价体系 |
| 语义图层 | 无 | 需新增 |
| 批量生成 | 无 | 需新增 |
| SVG 矢量导出 | `CanvasExport` | 需扩展 |
