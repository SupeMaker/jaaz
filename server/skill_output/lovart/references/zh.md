## Advanced AI Editing

**URL:** https://www.lovart.ai/zh/docs/edit-your-design/advanced-ai-editing

**Contents:**
- Advanced AI Editing
  - Quick Edit
  - Mark / Touch Edit
  - Edit Elements
  - Edit Text

Quick Edit provides fast access to common editing tasks. Pick a frequently used adjustment from the panel, or prompt for a custom edit.

Shortcut: M or Cmd/Ctrl+Click

Touch Edit lets you mark an element in an image and describe a precise edit in natural language. Lovart identifies the element and applies the edit.

Edit Elements breaks your image into individual layers, so you can reposition, resize, and adjust subjects, backgrounds, and text independently. Use it when you want to manually rearrange the composition.

Edit Text lets you modify text within an image while preserving the original font and visual style. Use it when you only need to change the words, for example swapping a headline on an AI-generated poster or correcting a typo on a product label.

## AI Transformation

**URL:** https://www.lovart.ai/zh/docs/edit-your-design/ai-transformation

**Contents:**
- AI Transformation
  - Multi-Angles
  - Move Object
  - Vectorize
  - Mockup

Multi-Angles takes one image and produces variants of the same subject from different viewpoints. Instead of prompting, you drag or adjust parameters in a panel. Use it for product listings, character sheets, video storyboards, and any image set that needs to show a subject from multiple angles.

Multi-Angles has two modes:

Move Object moves one object in an image to a new position while preserving the original background. Select the object with the Rectangle or Lasso tool, then drag it to the desired location. Lovart generates a coherent new image with the original location filled in.

Vectorize turns a raster image into an SVG by tracing its contours and shapes as multi-path vector geometry. Use it when you need a logo, icon, or illustration that stays crisp at any size and exports cleanly to design or print tools.

Mockup shows a flat design on a real-world object. Start from a photo of the object you want to mock — a poster, box, T-shirt — then place and position your design on it. Use it to present logos, labels, posters, or UI screens in context.

## Basic AI Editing

**URL:** https://www.lovart.ai/zh/docs/edit-your-design/basic-ai-editing

**Contents:**
- Basic AI Editing
  - Upscale
  - Remove Background
  - Eraser
  - Expand

Upscale boosts the resolution of an image to 2K, 4K, or 8K. Use it to prepare assets for print, large displays, or high-resolution exports.

Remove BG detects the main subject in an image and replaces the background with transparency. Useful for clean product shots, isolating subjects, or preparing overlays.

Eraser removes unwanted objects, people, text, or watermarks from an image and fills the gap with AI-generated content that matches the surrounding scene.

Expand extends an image beyond its current bounds and fills the new area with AI-generated content that coherently completes the scene. Use it for outpainting, aspect ratio changes, extra breathing room, and adapting designs to different formats.

## Basic Editing

**URL:** https://www.lovart.ai/zh/docs/edit-your-design/basic-editing

**Contents:**
- Basic Editing
  - Adjust
  - Crop
  - Flip
  - Rotate

Adjust is a panel for basic image editing, covering light, exposure, vibrance, saturation, vignette, and other look controls. Use it when you want quick manual slider tuning.

Crop lets you reframe an image by keeping only the area you want. Drag to define the region, enter exact width and height values, or pick a preset aspect ratio.

Flip mirrors an image across its horizontal or vertical axis.

Rotate turns an image by a chosen angle, either in 90° increments or by a custom value.

## Canvas Objects

**URL:** https://www.lovart.ai/zh/docs/edit-your-design/canvas-objects

**Contents:**
- Canvas Objects
  - Add to Canvas
  - Frame
  - Shapes
  - Pencil
  - Text

Import your images and videos directly onto the Canvas for selection and editing.

Frames act as containers on the infinite Canvas — use them to structure multi-panel layouts, group related elements, or define artboards for specific export sizes. Add images, videos, text, or shapes inside a frame to keep related content together.

Shapes are flexible building blocks for your layout. Use them as masks, backgrounds, or graphic elements.

Draw freehand vector lines on the Canvas with the Pencil tool.

The Text tool places editable text layers on the Canvas — headlines, body copy, labels, and more. Each layer sits independently on top of your design and supports font selection, sizing, weight, alignment, and hierarchy.

## Chat Tools

**URL:** https://www.lovart.ai/zh/docs/how-to-prompt/chat-tools

**Contents:**
- Chat Tools
  - Mention
  - Web Search
  - Thinking & Fast Mode
  - Voice Mode
  - OpenClaw
  - Chat History

Type @ in your prompt to mention AI models, Brand Kits, or elements on the Canvas. The Agent will use only those resources as context for the current conversation. Unlike Select AI Models, which sets a soft preference, an @ mention is a high-priority instruction that overrides preferences.

Web Search integrates real-time web data into your design context, allowing the Agent to reference up-to-date information, current trends, visual inspiration, and design workflows. You can trigger it by clicking the icon, typing a search prompt, or pasting a public URL.

Method 1: Turn on Web Search

Method 2: Prompt for Semantic Search

Method 3: Drop a Public Web Link

Lovart offers two modes that control how the Agent approaches a request. In Thinking Mode, the Agent reasons longer and returns comprehensive, structured solutions. In Fast Mode, the Agent generates multiple creative options quickly for rapid exploration.

Voice Mode lets you communicate with the Agent by speaking — generate, edit, and ask questions with your voice.

Install the Lovart Skill into your OpenClaw environment, authenticate with your Lovart keys, and chat with your Lovart Agent wherever your OpenClaw runs. You can then generate AI images directly in your OpenClaw workspace using Lovart's creative Agent.

See the Lovart OpenClaw User Guide for details.

Chat History lists every session you've had inside a project, so you can revisit a past conversation.

## Customize Your Canvas

  - Hand
  - Canvas Background

The Select tool lets you pick, move, resize, and manage design elements on the infinite Canvas.

## Other Things You Can Prompt to Create

**URL:** https://www.lovart.ai/zh/docs/how-to-prompt/other-things-you-can-prompt-to-create

**Contents:**
- Other Things You Can Prompt to Create
  - Font Generator
  - Lovart Slides
  - Video Clipper

Font Generator produces a full custom font from a reference image, a text description, or both. Use it for stylized poster titles, e-commerce hero images, branded display type, or any time you need typography that doesn't exist in a stock font library. Generated fonts are saved to your Font Library and can be applied to any text layer on the Canvas.

Lovart Slides generates a full presentation deck from text, reference images, or a source PDF. The Agent researches and organizes the content, plans the deck sequence, and keeps the visual language coherent across pages.

Video Clipper combines multiple video clips, images, audio tracks, and text into a single edited video. Use it when you need to assemble a video from multiple sources.

## Selecting AI Models

**URL:** https://www.lovart.ai/zh/docs/how-to-prompt/selecting-ai-models

**Contents:**
- Selecting AI Models
  - Select AI Models
  - Image Generator
  - Video Generator

Open the Models panel to set model preferences for the current task. Turn Auto On to let the Agent choose the most suitable model automatically. Turn Auto Off to manually select the image, video, or 3D models you want to prioritize. The Agent may still route between similar models to balance quality, speed, and stability.

Image Generator runs your prompt with an image model directly. Use it when you already know the model, resolution, and aspect ratio you want and don't want the Agent to pick for you.

Video Generator runs your prompt and input media through a video model directly to create engaging videos. Use it when you want fast, direct control over the model and settings.
